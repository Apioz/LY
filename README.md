# 中台系统 - 安全检查模块交互原型

基于提供的设计图实现的可交互前端原型，包含完整侧边栏菜单、顶部 Tab 切换、筛选与时间区间切换。

## 功能模块

### 一级菜单：安全检查
- 安全检查统计（图表 + 集团/物业、月/季/年筛选）
- 检查管理：检查日历、检查项目设置、检查内容录入、检查点位设定、检查项目设定、检查计划设定、检查工单管理、全部检查
- 验收工单、整改工单、检查报表

### 一级菜单：告警中心
- 告警统计、告警列表、告警设置

### 一级菜单：设施工单

### 一级菜单：系统管理
- **检查内容配置**：列表与新增/编辑/查看均含 **安全等级** 字段
- 系统字典

## 启动

```bash
npm install
npm run dev
```

浏览器访问 http://localhost:5173

## 技术栈

React 18 + Vite + TypeScript + Ant Design 5 + ECharts
